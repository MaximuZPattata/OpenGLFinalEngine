# I have created a soft body simulation and handled each soft body with each thread. I can also manipulate the thread to handle more than 1 soft body at a time. I have demonstrated it in the video. Please check it out !

## Controls - There are camera controls for the user.
	 - W,A,S,D,Q,E for camera movement
	 - Mouse movement for camera rotation
	 - Ctrl + Z /Ctrl + X to change models 
	 - Ctrl + B for wireframe mode
	 - Spacebar to pause and resume mouse mouvement(be default it is set to pause)
	 - 1, 2, 3, 4, 5 for camera angles
